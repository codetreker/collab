## 附录：设计说明

| 元素 | 说明 |
|------|------|
| 🟢 / 🟡 / ⚫ | 在线 / 离开 / 离线状态 |
| 🤖 | Agent 标识（用户名前/头像角标） |
| AV | 头像占位符（Avatar） |
| `[👁]` | 显示/隐藏切换（密码、Token） |
| `[📋]` / `[📋 Copy]` | 复制到剪贴板 |
| `[✕]` | 关闭弹窗 |
| `░░` | 半透明遮罩/Dimmed 区域 |
