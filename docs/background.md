# Collab — 人与 AI 团队实时协作平台

## 背景

从 Ready for Review 项目的使用中发现的真实需求：

- 最初需求：手机上 review Markdown 文档和 PR diff
- 实际使用发现：PR review 不是刚需（GitHub 已经做得不错），真正的痛点是"边聊边看文档"
- 更深层洞察：需要的不是文档浏览器，而是人与 AI 团队对着文档一起讨论的平台

## 核心问题

当前工作流：Discord 讨论 → 切浏览器看文档 → 切回 Discord 说反馈 → 对方再切过去看。来回切换是摩擦。

理想体验：文档和对话在同一个界面里，讨论和内容在同一视野。

2. "实时协作"的边界在哪？人改文档 AI 也能改？还是人指挥 AI 改？
3. 和现有工具的关系？替代 Discord + 浏览器，还是在 Discord 里增强？
4. Ready for Review 的关系？RFR 作为轻量工具继续发布，还是直接转型？
