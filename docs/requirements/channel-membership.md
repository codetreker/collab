# 频道成员管理 — PRD

日期：2026-04-19 | 状态：Approved | 作者：野马（PM）

---

## 背景

Collab 当前所有用户自动加入所有频道，没有成员管理机制。随着团队规模增长和讨论场景分化，需要支持"不是所有 channel 对所有人可见"——即频道可见性控制和成员管理能力。

## 目标用户

- **管理员**：需要创建私密讨论空间，控制谁能看到和参与特定频道
- **普通用户**：需要清晰的频道列表，只看到与自己相关的频道
- **新用户**：注册后应有合理的默认频道体验，不被无关频道淹没

## 核心需求

### 需求 1: 频道可见性

频道分为**公开（public）**和**私有（private）**两种类型：

| 类型 | 可见性 | 加入方式 |
|------|--------|----------|
| 公开（public） | 所有用户可见 | 自由加入 |
| 私有（private） | 仅成员可见 | 需被邀请 |

**规则：**
- `#general` 始终是公开频道，不可改为私有
- 已有频道默认为公开（向后兼容）
- 频道类型在创建后可修改（public ↔ private），修改时需确认提示

**用户故事：**
> 作为管理员，我想创建只有特定成员可见的频道，以便进行私密讨论。

**验收标准：**
- [ ] 创建频道时可选择 public / private
- [ ] 私有频道只在成员的侧边栏显示
- [ ] 非成员看不到私有频道的任何信息（包括频道名称、消息、成员列表）
- [ ] `#general` 无法被设置为私有（UI 禁用该选项）
- [ ] 已有频道迁移后默认为 public，无数据丢失

### 需求 2: 成员管理

频道创建者和管理员可以添加或移除成员。

**添加成员：**
- 从用户列表中搜索并选择
- 支持一次添加多个用户
- 被添加的用户立即在侧边栏看到该频道

**移除成员：**
- 在频道设置的成员列表中操作
- 频道创建者和管理员可移除成员
- 被移除的成员立即看不到该频道

**用户故事：**
> 作为频道创建者，我想管理谁可以参与这个频道的讨论，以便保持讨论的相关性和私密性。

**验收标准：**
- [ ] 频道设置里有成员列表，显示所有当前成员
- [ ] 可以添加新成员（支持搜索/选择用户）
- [ ] 可以移除成员（频道创建者和管理员可操作）
- [ ] 被移除的成员立即看不到该频道
- [ ] 成员变更实时生效，无需刷新页面
- [ ] 公开频道也展示成员列表（但所有用户可自行加入/离开）

### 需求 3: 新用户默认频道

新用户注册时的频道分配逻辑：

- 自动加入所有**公开频道**（`#general` 等）
- **不**自动加入任何私有频道
- 私有频道需由频道管理员手动邀请

**用户故事：**
> 作为新加入团队的用户，我希望自动看到所有公开频道，而不会被拉入不相关的私有讨论。

**验收标准：**
- [ ] 新用户注册后只看到公开频道
- [ ] 新用户自动成为所有公开频道的成员
- [ ] 私有频道需要被手动邀请才可见
- [ ] 新创建的公开频道，已有用户自动加入（保持当前行为一致性）

## 不在 v1 范围

以下功能明确排除在本次迭代之外，留待 v2 考虑：

- **频道角色/权限分级**（如 owner / moderator / member 区分）— v2
- **频道邀请链接**（生成可分享的链接邀请成员）— v2
- **频道申请加入流程**（非成员可请求加入，管理员审批）— v2
- **频道归档/只读模式** — v2
- **批量频道管理**（管理员一次性设置多个频道）— v2

## 成功指标

- 私有频道创建数 > 0（功能被使用）
- 非成员无法通过任何路径访问私有频道内容（安全性）
- 新用户首次登录时频道列表符合预期（体验一致性）

## 开放问题

| # | 问题 | 状态 | 决策 |
|---|------|------|------|
| 1 | Agent（机器人）是否自动加入所有频道（包括私有）？还是需要手动添加？ | ✅ 已关闭 | Agent 不自动加入私有频道，和人一样需要手动添加 |
| 2 | 频道创建者离开频道后，谁有管理权限？（v1 是否需要处理这个边界情况？） | ✅ 已关闭 | Admin 角色有所有频道管理权；v1 不做 channel owner 概念 |
| 3 | 公开频道改为私有时，已有成员是否全部保留？ | ✅ 已关闭 | 保留已有成员 |
| 4 | 私有频道改为公开时，是否所有用户自动加入？ | ✅ 已关闭 | 所有用户自动加入 |

**补充决策：** Admin 可以看到和管理所有私有频道，即使自己不是成员。
