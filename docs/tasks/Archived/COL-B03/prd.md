# COL-B03: 公开频道预览 + 申请加入 — PRD

日期：2026-04-21 | 状态：Draft

## 背景
公开频道目前对未加入的用户不可见。需要让用户能预览公开频道内容并自行加入。

## 需求
1. 侧边栏显示用户未加入的公开频道（灰色/半透明，和已加入的区分）
2. 点击未加入的公开频道，显示最近 24h 消息（只读预览）
3. 预览模式下输入框替换为"加入频道"按钮
4. 点击"加入频道"后立即加入（v1 自动通过，不需审批）
5. 加入后输入框恢复正常，可以发消息
6. 私有频道不受影响（不在侧边栏显示给非成员）

## 验收标准
- [ ] 未加入的公开频道在侧边栏可见（样式区分）
- [ ] 点击后显示 24h 消息预览
- [ ] 预览模式无法发消息
- [ ] 点击"加入频道"后立即可发消息
- [ ] 私有频道不显示给非成员

## Agent 行为
- Agent 不自动加入公开频道（需要 owner 操作）
- Agent 的公开频道预览通过 API 实现（GET /api/v1/channels/:id/preview）

## 不在范围
- 审批流程（v2）
- 频道搜索/发现页
- 频道推荐

## 成功指标
- 用户通过预览加入频道的转化率 > 50%
