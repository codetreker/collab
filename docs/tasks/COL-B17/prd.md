# COL-B17: @mention 键盘过滤 — PRD

日期：2026-04-22 | 状态：Draft

## 背景
当前输入框输入 `@` 时没有任何提示。用户需要手动输入完整用户名。需要做一个 @mention 下拉过滤器，输入 `@` 后弹出成员列表，键入字符实时过滤，选中后插入格式化的 mention。

## 目标用户
Collab 所有用户（人类和 Agent）

## 核心需求

### 需求 1: @mention 触发
- 用户故事：作为聊天参与者，我想输入 `@` 后看到当前频道成员列表，以便快速 mention 某人
- 验收标准：
  - [ ] 输入 `@` 弹出下拉列表，显示当前频道成员
  - [ ] 列表包含用户名 + 头像（或 Agent 图标）
  - [ ] 按用户名和显示名匹配过滤

### 需求 2: 键盘过滤
- 用户故事：作为用户，我想边打字边缩小候选范围
- 验收标准：
  - [ ] `@ji` 过滤出 "jianjun" 等匹配项
  - [ ] 支持拼音首字母和英文名匹配
  - [ ] 无匹配时显示"无结果"

### 需求 3: 选中插入
- 用户故事：作为用户，我想点击或按 Enter 选中成员，自动插入 mention
- 验收标准：
  - [ ] 点击或 Enter 选中，插入 `@用户名` 并高亮显示
  - [ ] mention 作为不可编辑的整体（删除时整体删除）
  - [ ] 发送后服务端识别 mention 并存储关联

### 需求 4: 键盘导航
- 验收标准：
  - [ ] ↑/↓ 在候选列表中移动
  - [ ] Enter 选中当前项
  - [ ] Esc 关闭下拉

## 不在范围
- @everyone / @here 全员 mention — v2
- mention 推送通知 — v2
- 频道 mention（#channel）— v2

## 成功指标
- 用户 mention 使用率提升
