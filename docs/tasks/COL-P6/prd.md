# P6: 斜杠命令（Slash Commands）— PRD

日期：2026-04-21 | 状态：Approved

## 背景
用户在输入框输入 `/` 时，弹出命令列表，选择后执行对应操作。类似 Discord/Telegram 的 slash commands。这为 Human-Agent 协作提供了统一的命令接口——人类用户可以通过命令触发操作，Agent 也可以通过 API 注册自定义命令。

## 目标用户
- 人类用户：快速执行常用操作（不用找按钮）
- Agent owner：为自己的 Agent 注册自定义命令（v2）

## 功能需求

### 核心交互
- 输入框输入 `/` 时弹出命令列表（自动补全面板）
- 输入更多字符时过滤匹配的命令
- 选择命令后：
  - 简单命令直接执行
  - 带参数的命令显示参数输入提示
- 按 Esc 关闭面板
- 按 Tab/Enter 选中当前高亮命令

### 内置命令（v1）
1. `/help` — 显示所有可用命令
2. `/invite @user` — 邀请用户到当前频道
3. `/leave` — 离开当前频道
4. `/topic <text>` — 设置频道主题/描述
5. `/dm @user` — 打开与某人的 DM

### v2 预留命令
- `/status <text>` — 设置个人状态（需要后端状态存储）
- `/clear` — 清空聊天（需要确认交互 + 定义清空语义）
- `/nick <name>` — 修改显示名

### 命令注册机制
- v1：内置命令硬编码
- v2：Agent 可通过 API 注册自定义命令（如 `/deploy`、`/search`）

### UI 设计
- 命令面板样式参考 Discord：
  - 深色背景圆角弹窗
  - 每个命令显示：名称 + 简要描述
  - 高亮当前选中项
  - 键盘上下键导航
- 输入参数时显示参数名和类型提示

### 验收标准
- [ ] 输入 `/` 弹出命令列表
- [ ] 输入字符过滤命令
- [ ] 选中命令执行对应操作
- [ ] 至少 5 个内置命令可用
- [ ] Tab/Enter 选中，Esc 关闭
- [ ] 无命令匹配时显示"没有找到命令"

## 不在范围
- Agent 自定义命令注册（v2）
- 命令权限控制（v2）
- 命令历史/最近使用
- Bot 命令（Agent 通过消息触发，不通过 slash command）

## 成功指标
- 用户使用命令的频率 > 20%（相比纯 UI 操作）
