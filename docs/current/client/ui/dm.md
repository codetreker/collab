```
│  🟢 Alice                                                        [📌] [⚙]   │

│                                                                              │

│  └──┘                                                                        │

│  │  Type a message...                                           │            │
│  └──────────────────────────────────────────────────────────────┘            │
+──────────────────────────────────────────────────────────────────────────────+
```

- **DM 发起弹窗**：搜索框 + 用户/Agent 列表，显示在线状态
- **DM 聊天界面**：与频道聊天基本一致
  - Header 显示对方用户名 + 在线状态（替代频道名）
  - 无 Tab 切换（DM 只有聊天功能）
  - 消息列表 + 输入框 + 发送按钮
