# 10. Slash Commands (B07 / B08)

> Agent 自定义命令 + 内置命令的交互线框图。

## 10a. 命令面板（`/` 触发）

用户在输入框键入 `/` 后，弹出命令面板浮层。

```
+────────────────────+─────────────────────────────────────────────────────────+
│ (sidebar)          │  # general                                    [⚙]  [📌] │
│                    ├─────────────────────────────────────────────────────────┤
│                    │                                                         │
│                    │  ┌──┐  Alice           10:30 AM                        │
│                    │  │AV│  Hey, can someone deploy?                         │
│                    │  └──┘                                                   │
│                    │                                                         │
│                    │                                                         │
│                    │                                                         │
│                    │                                                         │
│                    ├ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┤
│                    │  ┌───────────────────────────────────────────────────┐  │
│                    │  │  ⚡ Slash Commands                          [✕]  │  │
│                    │  ├───────────────────────────────────────────────────┤  │
│                    │  │                                                   │  │
│                    │  │  ── System ──────────────────────────────────     │  │
│                    │  │  /status      Show channel info & members         │  │
│                    │  │  /clear       Clear chat history                  │  │
│                    │  │  /nick        Change your display name            │  │
│                    │  │                                                   │  │
│                    │  │  ── 🤖 DeployBot ────────────────────────────     │  │
│                    │  │  /deploy      Deploy to production                │  │
│                    │  │  /rollback    Rollback last deployment            │  │
│                    │  │  /logs        Show recent deploy logs             │  │
│                    │  │                                                   │  │
│                    │  │  ── 🤖 ReviewBot ────────────────────────────     │  │
│                    │  │  /review      Start code review                   │  │
│                    │  │  /deploy      Deploy to staging                   │  │
│                    │  │                                                   │  │
│                    │  └───────────────────────────────────────────────────┘  │
│                    ├─────────────────────────────────────────────────────────┤
│                    │  ┌─────────────────────────────────────────┐  [Send]    │
│                    │  │  /█                                      │            │
│                    │  └─────────────────────────────────────────┘            │
+────────────────────+─────────────────────────────────────────────────────────+
```

- **触发**：输入框键入 `/` 后立即弹出浮层
- **分组**：System（内置命令）在最上方，Agent 命令按 Agent 名称分组
- **每行**：命令名（加粗）+ 简短描述
- **Agent 组**：带 🤖 前缀的 Agent 名称作为分组标题
- **注意**：`/deploy` 同时出现在 DeployBot 和 ReviewBot 下（同名不冲突）

## 10b. 命令面板 — 搜索过滤

用户继续输入字符后，实时过滤匹配的命令。

```
  ┌───────────────────────────────────────────────────┐
  │  ⚡ Slash Commands                          [✕]  │
  ├───────────────────────────────────────────────────┤
  │                                                   │
  │  ── 🤖 DeployBot ────────────────────────────     │
  │  /deploy      Deploy to production     ← 匹配    │
  │                                                   │
  │  ── 🤖 ReviewBot ────────────────────────────     │
  │  /deploy      Deploy to staging        ← 匹配    │
  │                                                   │
  │  No more results.                                 │
  │                                                   │
  └───────────────────────────────────────────────────┘
  ┌─────────────────────────────────────────┐  [Send]
  │  /de█                                    │
  └─────────────────────────────────────────┘
```

- **实时过滤**：输入 `/de` 只显示命令名包含 "de" 的结果
- **保持分组**：过滤后仍按 Agent 分组；空组自动隐藏
- **System 组**：此例中无匹配项，整组隐藏

## 10c. 命令面板 — 同名命令选择

多个 Agent 注册了相同命令名时，用户需选择目标 Agent。

```
  ┌───────────────────────────────────────────────────┐
  │  ⚡ /deploy — Choose Agent                 [✕]  │
  ├───────────────────────────────────────────────────┤
  │                                                   │
  │  Multiple agents provide this command:            │
  │                                                   │
  │  ┌─────────────────────────────────────────────┐  │
  │  │  🤖 DeployBot                               │  │
  │  │  Deploy to production                       │  │
  │  └─────────────────────────────────────────────┘  │
  │                                                   │
  │  ┌─────────────────────────────────────────────┐  │
  │  │  🤖 ReviewBot                               │  │
  │  │  Deploy to staging                          │  │
  │  └─────────────────────────────────────────────┘  │
  │                                                   │
  └───────────────────────────────────────────────────┘
  ┌─────────────────────────────────────────┐  [Send]
  │  /deploy █                               │
  └─────────────────────────────────────────┘
```

- **触发**：用户选中一个同名命令（如 `/deploy`）时自动弹出
- **卡片列表**：每个 Agent 一张卡片，显示 Agent 名 + 该 Agent 的命令描述
- **点击卡片**：选定 Agent 后进入参数输入阶段

## 10d. 命令参数输入

选中命令后，面板切换为参数输入表单。

```
  ┌───────────────────────────────────────────────────┐
  │  ⚡ /deploy (DeployBot)                    [✕]  │
  ├───────────────────────────────────────────────────┤
  │                                                   │
  │  branch *                                         │
  │  ┌─────────────────────────────────────────────┐  │
  │  │  e.g. main                                  │  │
  │  └─────────────────────────────────────────────┘  │
  │                                                   │
  │  environment *                                    │
  │  ┌─────────────────────────────────────────────┐  │
  │  │  e.g. production                            │  │
  │  └─────────────────────────────────────────────┘  │
  │                                                   │
  │  message                                          │
  │  ┌─────────────────────────────────────────────┐  │
  │  │  Optional deploy note...                    │  │
  │  └─────────────────────────────────────────────┘  │
  │                                                   │
  │                             [Cancel]  [▶ Execute] │
  │                                                   │
  └───────────────────────────────────────────────────┘
  ┌─────────────────────────────────────────┐  [Send]
  │  /deploy █                               │
  └─────────────────────────────────────────┘
```

- **参数标签**：字段名，必填参数带 `*` 标记
- **Placeholder**：灰色提示文本（`e.g. main`、`Optional deploy note...`）
- **按钮**：Cancel 返回命令列表，Execute 发送命令
- **无参数命令**：跳过此步，直接发送（如 `/status`）

## 10e. 命令执行中 & 结果

命令发送后在消息流中的展示。

```
┌──┐  Alice                         10:35 AM
│AV│  ⚡ /deploy branch:main env:production
└──┘  ┌─────────────────────────────────────────────────────┐
      │  🤖 DeployBot  ·  executing...          ⏳         │
      │  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  │
      └─────────────────────────────────────────────────────┘
```

命令完成后更新：

```
┌──┐  Alice                         10:35 AM
│AV│  ⚡ /deploy branch:main env:production
└──┘  ┌─────────────────────────────────────────────────────┐
      │  🤖 DeployBot  ·  completed              ✅         │
      │                                                     │
      │  Deployed main to production                        │
      │  Commit: a1b2c3d                                    │
      │  Duration: 47s                                      │
      │                                                     │
      └─────────────────────────────────────────────────────┘
```

- **命令消息**：带 ⚡ 前缀，区别于普通消息
- **参数展示**：`key:value` 格式跟在命令名后
- **结果卡片**：Agent 名 + 状态（executing / completed / failed）
- **进行中**：显示进度条占位 + ⏳ 图标
- **完成后**：卡片原地更新，显示结果内容 + ✅ 或 ❌

## 10h. `/nick` 昵称修改

输入新昵称并显示修改结果。

### 输入阶段

```
  ┌───────────────────────────────────────────────────┐
  │  ✏️  Change Nickname                        [✕]  │
  ├───────────────────────────────────────────────────┤
  │                                                   │
  │  New nickname *                                   │
  │  ┌─────────────────────────────────────────────┐  │
  │  │  e.g. CoolAlice                             │  │
  │  └─────────────────────────────────────────────┘  │
  │                                                   │
  │  Current: Alice                                   │
  │                                                   │
  │                             [Cancel]  [✏️ Change] │
  │                                                   │
  └───────────────────────────────────────────────────┘
```

### 成功后系统消息

```
┌──┐  Alice                         10:45 AM
│AV│  ⚡ /nick CoolAlice
└──┘
      ┌─────────────────────────────────────────────────────┐
      │  ✅  Nickname changed: Alice → CoolAlice            │
      └─────────────────────────────────────────────────────┘
```

- **输入面板**：显示当前昵称，输入新昵称
- **系统消息**：修改成功后显示 before → after
- **仅发送者可见**：ephemeral 消息

## 附录：交互流程

```
用户输入 /
    │
    ▼
命令面板弹出（按 Agent 分组）
    │
    ├── 继续输入 → 实时过滤
    │
    ├── 选中命令
    │       │
    │       ├── 无同名 → 直接进入参数输入
    │       │
    │       └── 有同名 → 显示 Agent 选择
    │               │
    │               └── 选中 Agent → 进入参数输入
    │
    └── 按 Esc / 点击外部 → 关闭面板

参数输入
    │
    ├── 有参数 → 填写表单 → Execute
    │
    └── 无参数 → 直接发送

发送后
    │
    ├── 消息流显示 ⚡ 命令消息
    │
    ├── Agent 处理中 → executing... + ⏳
    │
    └── 完成 → 原地更新结果卡片（✅ / ❌）
```
