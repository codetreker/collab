- **频道 Header**：频道名 + 🔓 Public 标记
- **消息预览**：只读，展示最近几条消息，无法回复或 react
- **底部浮层**：半透明遮罩，显示频道统计（成员数 + 消息数）+ "Join Channel" 按钮
- **加入后**：浮层消失，切换为完整聊天视图，输入框可用
