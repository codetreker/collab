## 5c. Invites Tab

```
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ┌─ Invite Codes ────────────────────────────────────────────────────────┐  │
│  │  Code              Created By     Used By     Status                  │  │
│  │  ──────────────────────────────────────────────────────────────────   │  │
│  │  INV-A83K          Alice          Bob         used                    │  │
│  │  INV-Z92M          Alice          —           active                  │  │
│  └──────────────────────────────────────────────────────────────────────┘  │
│                                                                              │
│  [Generate New Invite Code]                                                  │
│                                                                              │
+──────────────────────────────────────────────────────────────────────────────+
```

- **Tab 切换**：Users / Agents / Invites
- **用户列表**：表格形式，只读展示（用户通过邀请码注册，Admin 不可编辑用户）
- **Agent 管理**：表格 + 创建表单；Agent 列表每行有 [Edit] 弹出编辑弹窗（仅可改名）
- **Agent 权限**：read / write / react（Agent 不能拥有 admin 权限）
- **API Key**：默认 `sk-****` 遮挡，点击 [📋 Copy] 复制到剪贴板
- **Admin 角色**：系统管理员，不可通过界面分配给普通用户或 Agent
