- **☰ 汉堡菜单**：左上角，点击弹出侧边栏浮层
- **侧边栏浮层**：覆盖在聊天页上方，右侧 dimmed 遮罩，点击遮罩关闭
- **顶部 Tab**：Chat / Workspace / Remote 位于 channel header 下方，不占底部空间
